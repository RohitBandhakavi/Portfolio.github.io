Nashville Housing Dataset
Data Cleaning and Transformation in SQL

This SQL script demonstrates a comprehensive data cleaning and transformation process on the Nashville housing dataset. It includes date formatting, filling in missing property addresses using self-joins, splitting combined address fields into separate components, standardizing categorical values, removing duplicates with ROW_NUMBER(), and dropping unnecessary columns. The goal is to prepare the dataset for accurate analysis and reporting through clean, well-structured data.
